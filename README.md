SNAKEN (SNAKE ENgine) is a library aimed at generalizing the famous "Snake" game in order to use it as a super simple world toy-simulation.<br/>

## Shared library installation (Linux)
Without graphics:<br/>
`make install`<br/>

With graphics:<br/>
`make install GRAPHICS=1`<br/>
